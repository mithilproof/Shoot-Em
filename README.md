# Shoot-Em

Shoot-Em is a space themed shoot 'em up where you must blast away unrelenting enemies before they destroy you. The game features 13 enemy types, 5 powerups, parallax backgrounds, retro sound effects, and locally stored stats.

# Controls

**Move:** WASD/Arrows  
**Aim/Fire:** Mouse  
**Autofire:** F  
**Pause:** P  
**Mute:** M
